Link Automation Script

The linkAutomation.js script checks the link for a variety of characteristics on an anchor tag. It checks what kind of icon the link needs and whether or not it needs a target attribute to force the link to open in a new tab. This means one can keep the code clean by not hard coding classes and targets on to anchor tags.
